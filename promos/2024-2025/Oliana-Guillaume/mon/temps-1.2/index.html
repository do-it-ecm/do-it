1. Optimisation de Portefeuille

1.1 Concepts de base

L'optimisation de portefeuille est une m√©thode qui permet de s√©lectionner les meilleurs actifs pour maximiser le rendement attendu tout en minimisant le risque. Le principe fondamental est bas√© sur la th√©orie moderne du portefeuille d√©velopp√©e par Harry Markowitz en 1952, connue sous le nom de mod√®le de Markowitz.

Concepts cl√©s :

Rendement attendu

Le rendement attendu est la moyenne des rendements possibles d'un portefeuille, pond√©r√©s par les probabilit√©s d'occurrence. Il est calcul√© comme une moyenne pond√©r√©e des rendements attendus de chaque actif en fonction de son poids dans le portefeuille.

$$ \text{Rendement attendu du portefeuille} = \sum_{i=1}^{n} w_i \times r_i $$

  • $w_i$ : Poids de l'actif $i$ dans le portefeuille (la proportion du capital total investi dans cet actif).
  • $r_i$ : Rendement attendu de l'actif $i$.

Volatilit√© (Risque)

La volatilit√© d'un portefeuille mesure l'incertitude quant aux rendements futurs. Elle repr√©sente l'√©cart-type des rendements d'un portefeuille et est une mesure du risque.

La volatilit√© d'un portefeuille d√©pend de la variance et de la covariance des actifs qui le composent. La variance mesure la dispersion des rendements d'un actif par rapport √† sa moyenne, tandis que la covariance mesure comment deux actifs se comportent ensemble.

$$ \text{Volatilit√© du portefeuille} = \sqrt{ \sum_{i=1}^{n} \sum_{j=1}^{n} w_i w_j \times \text{Cov}(r_i, r_j) } $$

  • $\text{Cov}(r_i, r_j)$ : Covariance entre les rendements de l'actif $i$ et l'actif $j$.

Ratio de Sharpe

Le ratio de Sharpe est une mesure de la performance d'un portefeuille ajust√©e au risque. Il compare le rendement exc√©dentaire (au-dessus du taux sans risque) √† la volatilit√© du portefeuille. Plus le ratio de Sharpe est √©lev√©, meilleure est la performance ajust√©e au risque.

$$ \text{Ratio de Sharpe} = \frac{\text{Rendement du portefeuille} - \text{Taux sans risque}}{\text{Volatilit√© du portefeuille}} $$

  • Rendement du portefeuille : Rendement moyen du portefeuille.
  • Taux sans risque : Rendement d'un actif sans risque (par exemple, les obligations d'√âtat).
  • Volatilit√© du portefeuille : √âcart-type des rendements du portefeuille.

Fronti√®re efficiente

La fronti√®re efficiente est un graphique montrant les portefeuilles qui offrent le meilleur rendement possible pour un niveau de risque donn√©. Chaque point sur la fronti√®re repr√©sente un portefeuille optimal, et tout portefeuille en dessous de la fronti√®re est consid√©r√© comme sous-optimal car il pr√©sente un rendement inf√©rieur pour un niveau de risque √©quivalent.

1.3 Optimisation avec la m√©thode de Markowitz

La m√©thode de Markowitz, √©galement appel√©e th√©orie moderne du portefeuille, est une technique qui permet de s√©lectionner les actifs dans un portefeuille afin de maximiser le rendement attendu tout en minimisant le risque (volatilit√©). Elle repose sur la relation entre la variance (ou volatilit√©) et les rendements attendus des actifs.

L'objectif principal est de d√©terminer la fronti√®re efficiente, c'est-√†-dire un ensemble de portefeuilles optimaux offrant le meilleur rendement pour un niveau de risque donn√©.

Formules cl√©s :

1. Rendement attendu du portefeuille

Le rendement attendu du portefeuille est la somme des rendements attendus pond√©r√©s de chaque actif :

$$ \text{Rendement attendu du portefeuille} = \sum_{i=1}^{n} w_i \times r_i $$

  • $w_i$ : Poids de l'actif $i$ dans le portefeuille (la proportion du capital investi dans cet actif).
  • $r_i$ : Rendement attendu de l'actif $i$.

2. Risque (volatilit√©) du portefeuille

Le risque (volatilit√©) d'un portefeuille est d√©termin√© √† partir de la variance des actifs individuels ainsi que de leurs covariances. Il est exprim√© par :

$$ \text{Volatilit√© du portefeuille} = \sqrt{\sum_{i=1}^{n} \sum_{j=1}^{n} w_i w_j \times \text{Cov}(r_i, r_j)} $$

  • $\text{Cov}(r_i, r_j)$ : Covariance entre les rendements des actifs $i$ et $j$.

3. Optimisation du portefeuille

L'objectif de l'optimisation du portefeuille est de minimiser la volatilit√© pour un rendement donn√© ou de maximiser le rendement pour un niveau de risque donn√©.

La fronti√®re efficiente est un ensemble de portefeuilles optimaux qui fournissent le meilleur rendement pour un niveau de risque donn√©. Chaque point sur cette fronti√®re repr√©sente un portefeuille optimal.

Code Python pour l'optimisation du portefeuille

On va ici chercher √† optimiser le portefeuille, c'est-√†-dire le poids des actifs selon les crit√®res de volatilit√©, rendement et risque.

Avec le code python suivant, on obtient les figures ci-dessous :



import yfinance as yf
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import scipy.optimize as sco
from tabulate import tabulate

# S√©lection des actions
actions = ['AAPL', 'MSFT', 'GOOGL']

# T√©l√©chargement des donn√©es
data = yf.download(actions, start="2020-01-01", end="2023-01-01")['Adj Close']

# Calcul des rendements quotidiens
rendements = data.pct_change().dropna()


jours_bourse_par_an = 252

# Calcul du rendement moyen annuel
rendement_moyen_annuel = rendements.mean() * jours_bourse_par_an * 100

# Calcul de la volatilit√© annuelle
volatilite_annuelle = rendements.std() * np.sqrt(jours_bourse_par_an) * 100
# Taux sans risque (par exemple, 1 % par an)
taux_sans_risque_annuel = 0.01  # 1% par an

# Calcul du ratio de Sharpe annuel
ratio_sharpe_annuel = (rendements.mean() * jours_bourse_par_an - taux_sans_risque_annuel) / (rendements.std() * np.sqrt(jours_bourse_par_an))

# Cr√©ation d'un DataFrame avec les r√©sultats annuels
resultats_annuels = pd.DataFrame({
    'Rendement Moyen Annuel (%)': rendement_moyen_annuel,
    'Volatilit√© Annuelle (%)': volatilite_annuelle,
    'Ratio de Sharpe Annuel': ratio_sharpe_annuel
})

# Affichage du tableau avec pandas (utile dans un notebook)
print(resultats_annuels)

# Affichage du tableau avec tabulate
print("\nTableau r√©capitulatif des rendements moyens, volatilit√©s et ratios de Sharpe (annuels) :")
print(tabulate(resultats_annuels, headers='keys', tablefmt='pretty'))

# ============================
# Optimisation de portefeuille avec la m√©thode de Markowitz
# ============================

# Calcul du rendement annuel moyen et de la matrice de covariance
rendement_annuel = rendements.mean() * jours_bourse_par_an  # Rendement annuel
cov_matrix_annuelle = rendements.cov() * jours_bourse_par_an  # Covariance annuelle

# Nombre d'actions dans le portefeuille
nombre_actions = len(actions)


def objectif_volatilite(weights):
    return np.sqrt(np.dot(weights.T, np.dot(cov_matrix_annuelle, weights)))


def objectif_sharpe(weights):
    rendement_portefeuille = np.sum(weights * rendement_annuel)
    volatilite_portefeuille = np.sqrt(np.dot(weights.T, np.dot(cov_matrix_annuelle, weights)))
    ratio_sharpe_portefeuille = (rendement_portefeuille - taux_sans_risque_annuel) / volatilite_portefeuille
    return -ratio_sharpe_portefeuille  # Nous minimisons l'inverse pour maximiser le ratio de Sharpe


def objectif_rendement(weights):
    return -np.sum(weights * rendement_annuel)  # Nous minimisons l'inverse pour maximiser le rendement


# Contraintes : somme des poids doit √™tre √©gale √† 1
contraintes = ({'type': 'eq', 'fun': lambda x: np.sum(x) - 1})

# Limites sur les poids des actifs (entre 0 et 1)
limites = tuple((0, 1) for asset in range(nombre_actions))

# ============================
# Optimisation pour chaque objectif
# ============================

# 1. Minimisation de la volatilit√©
result_volatilite = sco.minimize(objectif_volatilite, nombre_actions * [1./nombre_actions], method='SLSQP', bounds=limites, constraints=contraintes)
poids_optimal_volatilite = result_volatilite.x

# 2. Maximisation du ratio de Sharpe
result_sharpe = sco.minimize(objectif_sharpe, nombre_actions * [1./nombre_actions], method='SLSQP', bounds=limites, constraints=contraintes)
poids_optimal_sharpe = result_sharpe.x

# 3. Maximisation du rendement
result_rendement = sco.minimize(objectif_rendement, nombre_actions * [1./nombre_actions], method='SLSQP', bounds=limites, constraints=contraintes)
poids_optimal_rendement = result_rendement.x

# ============================
# Visualisation des r√©partitions optimales pour chaque strat√©gie
# ============================

# R√©partition pour minimisation de la volatilit√©
plt.figure(figsize=(8, 8))
plt.pie(poids_optimal_volatilite, labels=actions, autopct='%1.1f%%', startangle=90, colors=['lightblue', 'lightgreen', 'coral'])
plt.title("R√©partition des actifs - Minimisation de la volatilit√©")
plt.axis('equal')  # Assure que le camembert est un cercle
plt.show()

# R√©partition pour maximisation du ratio de Sharpe
plt.figure(figsize=(8, 8))
plt.pie(poids_optimal_sharpe, labels=actions, autopct='%1.1f%%', startangle=90, colors=['lightblue', 'lightgreen', 'coral'])
plt.title("R√©partition des actifs - Maximisation du ratio de Sharpe")
plt.axis('equal')  # Assure que le camembert est un cercle
plt.show()

# R√©partition pour maximisation du rendement
plt.figure(figsize=(8, 8))
plt.pie(poids_optimal_rendement, labels=actions, autopct='%1.1f%%', startangle=90, colors=['lightblue', 'lightgreen', 'coral'])
plt.title("R√©partition des actifs - Maximisation du rendement")
plt.axis('equal')  # Assure que le camembert est un cercle
plt.show()

rendements_portefeuille = []
volatilites_portefeuille = []
poids_optimal = []

# G√©n√©ration des portefeuilles pour diff√©rents niveaux de rendement
for rendement_cible in np.linspace(rendement_annuel.min(), rendement_annuel.max(), 100):
    contraintes = ({'type': 'eq', 'fun': lambda x: np.sum(x) - 1},
                   {'type': 'eq', 'fun': lambda x: np.sum(x * rendement_annuel) - rendement_cible})
    result = sco.minimize(objectif_volatilite, len(actions) * [1./len(actions)], method='SLSQP', bounds=limites, constraints=contraintes)

    rendements_portefeuille.append(rendement_cible)
    volatilites_portefeuille.append(np.sqrt(np.dot(result.x.T, np.dot(cov_matrix_annuelle, result.x))))
    poids_optimal.append(result.x)

# ============================
# Trac√© de la fronti√®re efficiente
# ============================
plt.figure(figsize=(10, 6))
plt.plot(volatilites_portefeuille, rendements_portefeuille, marker='o', linestyle='-', color='b')
plt.title('Fronti√®re efficiente (Markowitz)')
plt.xlabel('Risque (Volatilit√©)')
plt.ylabel('Rendement attendu')
plt.grid(True)
plt.show()

# ============================
# Visualisation de la r√©partition optimale pour minimisation de la volatilit√©
# ============================
# Choisir un portefeuille optimal (par exemple, le 50√®me portefeuille pour un compromis interm√©diaire)
poids_optimal_volatilite = poids_optimal[50]

# Cr√©ation d'un camembert pour afficher la r√©partition optimale des actifs
plt.figure(figsize=(8, 8))
plt.pie(poids_optimal_volatilite, labels=actions, autopct='%1.1f%%', startangle=90, colors=['lightblue', 'lightgreen', 'coral'])
plt.title("R√©partition des actifs - Minimisation de la volatilit√©")
plt.axis('equal')
plt.show()
